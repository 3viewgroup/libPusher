Overview

A PTPusherPresenceChannel object represents a Pusher presence channel.

Presence channels build on the security of Private channels and expose the additional feature of an awareness of who is subscribed to that channel. This makes it extremely easy to build chat room and “who’s online” type functionality to your application.

Presence channel names always have the prefix of “presence-”.

Unlike the Pusher Javascript client API, PTPusherPresenceChannel does not use events to notify when members are added or removed. Instead, you should assign a presenceDelegate which will be notified of these events.
