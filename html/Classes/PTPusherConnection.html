Instance Methods

connect

Establishes a web socket connection to the Pusher server.

- (void)connect

Discussion

Establishes a web socket connection to the Pusher server.

The delegate will only be sent a didConnect message when the web socket receives a ‘connection_established’ event from Pusher, regardless of the web socket’s connection state.

send:

Sends an object over the web socket connection.

- (void)send:(id)object

Discussion

Sends an object over the web socket connection.

The object will be serialized to JSON before sending, so the object must be anything that can be converted into JSON (typically, any plist compatible object).
